# Pizza Sales Analysis Using SQL

## Project Overview

This project analyzes pizza sales data using SQL to uncover valuable business insights. The analysis focuses on revenue generation, customer ordering patterns, pizza performance, and category-wise sales trends.

## Objectives

* Calculate total revenue generated from pizza sales
* Analyze customer ordering behavior
* Identify top-selling pizzas
* Study category-wise sales performance
* Perform revenue-based analysis

## Skills Demonstrated

* SQL
* Joins
* Aggregate Functions
* GROUP BY
* Subqueries
* Window Functions
* Revenue Analysis
* Data Analysis

## Project Questions

### Basic Analysis

1. Total number of orders placed
2. Total revenue generated
3. Highest-priced pizza
4. Most common pizza size ordered
5. Top 5 most ordered pizza types

### Intermediate Analysis

6. Total quantity ordered by pizza category
7. Distribution of orders by hour
8. Category-wise pizza distribution
9. Average pizzas ordered per day
10. Top 3 pizza types based on revenue

### Advanced Analysis

11. Percentage contribution of each pizza type to revenue
12. Cumulative revenue analysis
13. Top 3 pizza types by revenue within each category

### Additional Analysis

14. Which day of the week generated the highest revenue?
15. What is the average order value?
16. Which pizza category has the highest average order quantity?
17. Which pizza generated the highest revenue on weekends?
